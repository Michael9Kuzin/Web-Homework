# Проект-задачка: “Мини-каталог на Node.js + AJAX”

Создать небольшое веб-приложение, которое демонстрирует работу Node.js
-  статического сервера,
-  динамических маршрутов, 
-  AJAX, а также AJAX через jQuery или prototype.js.

## Структура проекта

```
mini-catalog/
├─ server.js
└─ public/
   ├─ index.html
   ├─ style.css
   └─ app.js
```

---
## Чек-лист для проверки проекта

1. Запускается ли `node server.js` без ошибок.
2. Открывается ли `http://localhost:3000/` и видна страница.
3. При клике «Загрузить товары» появляется список товаров.
4. Переключение метода AJAX меняет способ запроса (проверьте network в DevTools).
5. Сервер корректно отвечает на `GET /api/products` JSON.


# **Лабораторная работа №3 : “Каталог автомобилей на Node.js + AJAX”**

##  **Цель работы**

Закрепить навыки:

* создания статического Node.js сервера,
* разработки динамического API,
* выполнения AJAX-запросов (XMLHttpRequest + jQuery),
* работы с JSON-данными на стороне клиента.

---

# **Описание задания**

Создать мини-веб-приложение **«Каталог автомобилей»**, которое выводит список машин, полученных с Node.js сервера через AJAX.

### Приложение должно:

### **1. Использовать Node.js для:**

* отдачи статических файлов (`index.html`, `style.css`, `app.js`);
* возврата списка машин по маршруту:

```
GET /api/cars
```

### **2. Показывать на странице:**

* переключатель метода AJAX (Native / jQuery),
* кнопку «Загрузить автомобили»,
* список автомобилей (название, марка, год, цена).

### **3. Реализовать два варианта AJAX-запроса:**

* через **XMLHttpRequest (native)**,
* через **jQuery AJAX**.
